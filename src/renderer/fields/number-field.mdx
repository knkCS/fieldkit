# Number

A numeric input field. Supports minimum/maximum bounds, step increments, and optional prepend/append labels.

## When to Use

Use **Number** for any numeric value (integers or decimals). For currency, set `step: 0.01` and use `prepend` or `append` for the currency symbol.

## Settings

| Setting   | Type     | Default | Description                                    |
|-----------|----------|---------|------------------------------------------------|
| `min`     | `number` | ‚Äî       | Minimum allowed value                            |
| `max`     | `number` | ‚Äî       | Maximum allowed value                            |
| `step`    | `number` | ‚Äî       | Step increment for the input spinner             |
| `prepend` | `string` | ‚Äî       | Text displayed before the input                  |
| `append`  | `string` | ‚Äî       | Text displayed after the input                   |

## Validation

`min` and `max` from settings are applied as Zod constraints. There is no separate validation object for number fields ‚Äî bounds are inferred from settings.

| Rule       | Source   | Description                                     |
|------------|----------|-------------------------------------------------|
| `min`      | settings | Minimum value (`z.number().min(...)`)             |
| `max`      | settings | Maximum value (`z.number().max(...)`)             |

## Zod Output

```ts
// No constraints
z.number().optional()

// With min/max
z.number().min(0).max(150)

// Required field ‚Äî note: required is NOT checked in Zod for numbers
z.number().min(0).max(150)
```

## Table Cell

Renders the numeric value via `NumberCell`. No formatting (no thousand separators, no currency symbols).

## Known Limitations

- `required` not checked in Zod schema ‚Äî a number field marked `required` has no `.min(1)` or similar check
- `prepend`/`append` settings are defined in the schema but not passed to the `NumberField` component
- No integer vs decimal distinction
- `step` is not validated in Zod (UI-only constraint on the spinner)
